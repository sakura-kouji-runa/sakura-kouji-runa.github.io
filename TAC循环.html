发表于 2024-03-28 分类于 生物化学 阅读次数： 本文字数： 1.4k

柠檬酸循环·

概念·

有氧条件下，PA 有氧分解，形成 CO2 和 H2O
此途径：1.柠檬酸循环 2.氧化磷酸化
$ \ce{ G ->[ 氧化脱羧 ] PA -> 乙酰CoA ->[ 氧化脱羧 ] CO_2, H_2O, 能量 } $
地点：线粒体基质


意义·

  • TAC之速度

    • 取决于 细胞 对 ATP 的需求
    • 取决于 细胞 对 中间产物 的需求

  • 两用代谢途径

    • 是 糖、脂肪、氨基酸 氧化所经 共同途径
    • 其 中间产物 是众多 生物合成 的前体
    • 和 EMP 共同构成 糖的有氧代谢途径 并供能

EMP与TAC间的连接·

PA -> Ac-CoA 的不可逆过程 是连接EMP与TAC 的纽带。是由 PA脱氢酶系E 催化的 四步8反应

  • （E1）丙酮酸 脱氢酶组分
  • （E2）二氢硫辛酰 转乙酰基酶
  • （E3）二氢硫辛酸 脱氢酶

  1. PA在 丙酮酸 脱氢酶组分 的作用下 与TPP 反应为 羟乙基TPP | CO2

    E1被磷酸化修饰后即失活
    受E2上结合的 激酶和 磷酸酶的作用（激活剂：钙离子）

  2. 羟乙基TPP在 二氢硫辛酰 转乙酰基酶 的作用下 与硫辛酸 反应为 乙酰二氢硫辛酰胺 | TPP

    E2被Ac-CoA抑制，CoA解除抑制

  3. 乙酰二氢硫辛酰胺在 二氢硫辛酰 转乙酰基酶 的作用下 与CoA 反应为 Ac-CoA | 二氢硫辛酰胺

  4. 二氢硫辛酰胺在 二氢硫辛酸 脱氢酶 的作用下 与FAD 反应为 硫辛酸 | FADH2

    砷化物作用于二氢硫辛酰胺
    E3被NADH抑制，NAD+解除抑制

  5. FADH2 与FADH+ 反应为 FAD | NADH


以上反应受严格调控

  • 产物抑制
  • 磷酸化 和 去磷酸化

TAC的反应过程·

  1. 乙酰CoA在 柠檬酸合酶 的作用下与 草酰乙酸 | H2O 反应为 HSCoA | 柠檬酸

    受 ATP NADH 琥珀酸CoA 酯酰CoA 调控

  2. 柠檬酸 可脱去H2O 形成 顺乌头酸 可结合H2O 形成异柠檬酸

    平衡时 90：4：6

  3. 异柠檬酸在 异柠檬酸脱氢酶 的作用下 与NAD+ | NADP+ 反应为 草酰琥珀酸 | NADPH | NADH | 2H+

    在细菌中，其由 异柠檬酸裂解酶 作用为 琥珀酸 | 乙醛酸

  4. 草酰琥珀酸在 异柠檬酸脱氢酶 的作用下 与H+ 反应为 a-KG | CO2

    激活剂：ADP，Mg2+，NAD+，异柠檬酸

  5. a-KG在 a-KG脱氢酶复合体 的作用下 与NAD+ | HSCoA 反应为 琥珀酸CoA | NADH + H+ | CO2

    受NADH，琥珀酸CoA，高能电荷 抑制
    受Ca2+ 激活

  6. 琥珀酸CoA在 琥珀酸CoA合成酶 的作用下 与GDP | Pi 反应为 琥珀酸 | GTP | HSCoA

  7. 琥珀酸在 琥珀酸脱氢酶 的作用下 与FAD 反应为 延胡索酸 | FADH2

    琥珀酸脱氢酶 含FAD，也含3种铁硫聚簇
    琥珀酸脱氢酶 是 TAC唯一嵌在Mit内膜 的酶
    竞争抑制剂：丙二酸
    TAC 3nd氧化反应

  8. 延胡索酸在 延胡索酸酶 的作用下 与H2O 反应为 苹果酸

  9. 苹果酸在 苹果酸脱氢酶 的作用下 与NAD+ 反应为 草酰乙酸 | NADH + H+

总反应与回补机制·

总反应·

$ \ce{ Ac-CoA + 3NAD^+ + FAD + GDP + Pi + 2H_2O -> 2CO_2 + 3NADH + FADH_2 + GTP + CoA + 2H^+ } $

  • 单向
  • 不需要氧，但是离开氧化磷酸化无法进行
  • Ac-CoA 上的 2C 加入TAC，由 脱羧作用 产生 2CO2
  • 共4步氧化反应，产生 3NADH + H+ / 1FADH2 / GTP
  • 经呼吸链 3NADH -> 7.5ATP， 1FADH2 -> 1.5ATP， 9ATP + 1GTP = 10ATP
  • 到此处，1G 产出 12.5 * 2 + 7 = 32ATP

回补机制·

  • 回补

    • 丙酮酸 羧化
    • PEP 羧化
    • 苹果酸 脱氢
    • 氨基酸 形成

  • 中间产物可作他用

    • a-KG -> Glu
    • Ac-CoA -> 卟啉环
    • 草酰乙酸 -> 天冬氨酸
