形容词变化

发表于 2022-09-14 更新于 2022-10-18 分类于 日语语法 阅读次数： 本文字数： 339

形容词变化·

汉字 + い结尾·

主要变化·

  • adj い　+　名词　（不变）
  • adj く　+　动词　（变为く）
  • adj く　+ て + adj/形动（表并列）
  • adj く　+　ない　（表否定）
  • adj か　+　った　（表过去）
  • adj け　+　れば　（表假定）

敬体变化·

adj い（音变） + ございます

  • あ段 + い

お段 + う + ございます

例:ありがたい　>>>　ありがとうございます

  • い段 + い

い段 尾音接 ゅ + う + ございます

例:美（うつく）しい　>>>　美しゅうございます

  • う段 + い

う段 + う + ございます

  • え段没有形容词
     
  • お段 + い

お段 + う + ございます

例：素人（しろうと）  玄人（くろうと）  仲人（なこうど）  若人（わこうど）  

汉字 + な结尾·

与名词变化保持一致
